# Canadian Consultants – Architects & Consultant Engineers

A clean, professional landing page for Canadian Consultants.

## File Structure

```
canadian-consultants/
├── index.html          ← Main page (GitHub Pages entry point)
├── css/
│   └── style.css       ← All styles
├── js/
│   └── main.js         ← Sidebar, search, animations
├── images/
│   └── logo.png        ← Company logo
└── README.md
```

## GitHub Pages Deployment

1. Create a new GitHub repository (e.g. `canadian-consultants`)
2. Upload all files keeping the folder structure intact
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch and `/ (root)` folder
5. Click **Save** — your site will be live at:
   `https://YOUR-USERNAME.github.io/canadian-consultants/`

## Local Development

Simply open `index.html` in a browser — no build tools required.
All paths are relative and work from the file system directly.
